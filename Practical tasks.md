**Практична 3: Проєктування навчального досвіду (Драфт сторіборду та планування оцінювання)**

*   **Завдання:** Команда створює та презентує розширений "кістяк" (драфт) сторіборду для всього курсу. Цей драфт має включати:
    1.  **Структуру для 5 уроків:** Для кожного уроку прописано назву, цілі, список назв секцій, тезовий опис кожної секції та ідеї щодо форматів контенту (позначки: "текст", "фліп-карти" тощо).
    2.  **Ідею фінального практичного завдання:** Короткий опис того, що саме слухач курсу має зробити, щоб продемонструвати отриману навичку.
    3.  **Тезовий опис фінального квізу:** Опис того, які теми або рівні знань (за таксономією Блума) буде перевіряти квіз.

**Практична 4: Фіналізація сторіборду та розробка контенту для оцінювання**

*   **Завдання:** Команда перетворює драфти на готові до розробки сценарії. Результатом є фіналізований сторіборд, що містить:
    1.  **Повністю написаний текст** для всіх 5 уроків.
    2.  **Готовий контент для фінального квізу:** Сформульовані питання та варіанти відповідей.
    3.  **Деталізовану концепцію фінального практичного завдання:** Розширена ідея, що тепер включає чітко прописані **критерії його оцінювання**.
    4.  ***(Завдання із зірочкою, на вибір):*** Або розробити сценарій для інтерактивної вправи "Сценарій", або створити 5 питань для самоперевірки (Knowledge Checks) для розміщення всередині уроків.

**Практична 5: Первинна збірка прототипу в Articulate Rise**

*   **Завдання:** Команда збирає в Articulate Rise прототип курсу, який включає:
    1.  **Увесь готовий контент:** Тексти уроків, питання квізу та всі заплановані інтерактивні елементи (таби, акордеони тощо).
    2.  **Фіналізований блок практичного завдання:** Детальна інструкція для слухача та чіткі критерії оцінювання для викладача.
    3.  **Заглушки (placeholders)** на місцях, де пізніше з'явиться графіка.
    4.  *Важливо: на цьому етапі фінальне технічне налаштування (відстеження прогресу, система оцінювання) ще **не виконується**.*

**Практична 6: Фіналізація курсу (Графіка та технічне налаштування)**

*   **Завдання:** Команда демонструє фіналізований прототип курсу, в якому:
    1.  **Інтегровано всю графіку** в єдиному стилі (ілюстрації, іконки, зображення для хотспоту).
    2.  **Виконано фінальне технічне налаштування** в Articulate Rise: налаштовано систему оцінювання для квізу та параметри відстеження прогресу.
